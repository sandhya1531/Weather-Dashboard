# Weather-Dashboard
This Weather Dashboard project is a responsive, user-friendly web app built with HTML, CSS, and vanilla JavaScript. It fetches real-time weather data and forecasts using the OpenWeatherMap API.Features include persistent dark/light mode, search history, clean UI with animations, and error handling‚Äîideal for planning daily activities or travel.

## Features

- Search weather by city name or use device geolocation
- Displays UV index and Air Quality Index (AQI)
- Persistent dark/light mode toggle with smooth UI animations
- Search history functionality for quick access to recent locations
- Responsive design for desktop, tablet, and mobile devices
- Error handling and user-friendly messaging for invalid inputs or API issues
- Clean, modern, and accessible user interface

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/weather-dashboard.git
    cd weather-dashboard
    ```
2. Open `index.html` in your preferred browser.

## API Setup

- Obtain a free API key from [OpenWeatherMap](https://openweathermap.org/api).
- Replace the placeholder `'YOUR_OPENWEATHERMAP_KEY'` in `script.js` with your actual API key.

## Usage

- Enter a city name in the search bar and click "Search" or press Enter.
